## Getting Started

### Dependencies (node_module) 설치

```bash
npm install -g pnpm # pnpm 미설치 경우에만 실행
pnpm install
```

### 로컬 환경에서 Next.js Dev Server 실행 (development 모드)

```bash
pnpm run dev
```

### 접속 URL

[http://localhost:3000](http://localhost:3000)

### Build (배포)

```bash
pnpm run build
```
